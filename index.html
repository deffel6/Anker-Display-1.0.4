Live-Daten deiner Solarbank auf dem 1.28″ Rund-Display – ganz ohne Home Assistant.

Flashen

Dieser Browser unterstützt kein Web Serial – bitte Chrome oder Edge am Computer verwenden. Web Serial ist hier nicht erlaubt – Seite über HTTPS öffnen.

Board per USB anschließen, Button klicken, Port auswählen.
Funktioniert mit Chrome und Edge (Desktop).

Neu in 1.0.4

  • Passwort-Anzeige (Auge-Symbol) in allen Eingabeformularen
  • Falscher Anker-Login? Korrektur direkt im Browser – kein Neu-Flashen mehr nötig
  • GPIO9-Reset entfernt (im Gehäuse ohnehin nicht zugänglich)

Ersteinrichtung

  1. Nach dem Flashen startet das Display im Setup-Modus
  2. Mit dem WLAN Anker-Display-Setup verbinden
  3. Im Browser 192.168.4.1 öffnen und WLAN- & Anker-Zugangsdaten eingeben
  4. Das Display zeigt eine IP-Adresse – diese im Heimnetz aufrufen und die Anlage auswählen

Login fehlgeschlagen? Das Display zeigt dann eine IP-Adresse an – dort einfach die Zugangsdaten korrigieren. Neu flashen ist nicht nötig.
